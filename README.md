# KasiCode Cohorts

Training courses for building apps with AI. Hosted on GitHub Pages.

## Live Site

https://peterken1.github.io/KasiCode-Cohorts/

## Structure

```
├── index.html                  ← Course catalog (landing page)
├── vibe-coding-101/            ← First cohort
│   ├── index.html              ← Course overview & enrollment
│   ├── day-1.html              ← Daily lesson (add as needed)
│   ├── day-2.html
│   └── ...
├── fullstack-with-ai/          ← Future course
├── ai-for-business/            ← Future course
└── README.md
```

## Adding a New Course

1. Create a new folder: `my-new-course/`
2. Add an `index.html` with the course overview
3. Add daily lesson files as needed
4. Link it from the main `index.html` catalog
5. Push to `main` — GitHub Pages deploys automatically

## Payments

Cohort payments are handled through KasiCode's main platform:
- Participants pay via Yoco (integrated in kasicode.app)
- They receive a promo code that activates the `cohort` tier (50 builds)
- Training content here is public — the value is live instruction + tool access
