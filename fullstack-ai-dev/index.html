What you build

A CRM that would cost R40,000 to commission.

Not a todo app. Not a counter. A real, deployable, AI-powered CRM you can use for your own business, sell to clients, or launch as a SaaS. Every feature built from fundamentals — so you understand every line.

🔐
Secure Authentication
Sign up, login, logout with Supabase Auth. Session management via httpOnly cookies. Protected routes enforced at the middleware level. RLS policies so users only ever see their own data.
Day 3
👥
Contacts — Full CRUD
Add, view, edit and delete contacts. Server Actions talking directly to PostgreSQL. Optimistic UI updates. Dynamic routes. The pattern you build once and apply everywhere.
Day 4
💼
Deals, Notes & Activity Log
Deals linked to contacts via foreign keys. Notes with a hover-to-reveal delete. An auto-logging activity feed that records every action — deal created, stage changed, note added — automatically.
Day 5
📊
Live Dashboard
Persistent sidebar navigation with active route highlighting. Dashboard home showing total contacts, open deals, and pipeline value — all queried from Supabase in parallel with Promise.all.
Day 6
🗂️
Drag-and-Drop Kanban
A full pipeline board with five stages. Drag a deal card and it updates in Supabase instantly. Optimistic UI so the card never snaps back. Error recovery rolls back if the server fails.
Day 7
✨
Claude AI — Two Features
A contact history summariser that reads your notes, deals, and activity and briefs you in 4 sentences. An email drafter that writes a stage-appropriate follow-up — tone changes with the deal stage. Both stream word by word.
